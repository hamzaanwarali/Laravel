# نظام إدارة الكروت النقطية

## وصف المشروع
موقع إلكتروني يعمل كنظام لإدارة كروت تحتوي على نقاط. يمكن للمستخدم بعد إدخال الكرت الحصول على نقاط، ثم لاحقاً استبدال تلك النقاط بمكافآت معينة.

## المستخدمون
- **مستخدم عادي**: يقوم بإدخال الكرت للحصول على النقاط واستبدال النقاط بمكافآت.
- **مدير**: لديه لوحة تحكم خاصة يستطيع من خلالها:
  - إضافة كروت جديدة
  - إنشاء مكافآت جديدة
  - تعديل الكروت والمكافآت
  - حذف الكروت والمكافآت

## التقنيات المستخدمة
- Laravel 12
- Breeze (للمصادقة وإدارة حسابات المستخدمين)

## المتطلبات الأساسية
- PHP >= 8.2
- Composer
- MySQL
- Node.js & NPM (لموارد Breeze الأمامية)

## خطوات التثبيت والتشغيل

```bash
# 1. استنساخ المشروع
https://github.com/hamzaanwarali/Laravel.git
cd [Laravel]

# 2. تثبيت حزم PHP
composer install

# 3. تثبيت حزم Node.js
npm install

# 4. نسخ ملف البيئة
cp .env.example .env

# 5. توليد مفتاح التطبيق
php artisan key:generate

# 6. إعداد قاعدة البيانات في ملف .env ثم تشغيل المايجريشن
php artisan migrate

# 7. تجميع الأصول الأمامية
npm run build

# 8. تشغيل السيرفر المحلي
php artisan serve
